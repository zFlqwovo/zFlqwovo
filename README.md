## 🛸 关于我 (The Signal)

我是一个靠**自学**和**瞎折腾**成长的开发者。比起枯燥的理论，我更喜欢直接上手拆解、逆向、或者搭建各种奇怪的服务（然后看着它们崩掉）。

现实中我可能有点社恐（INTJ/P?），不太擅长应付复杂的社交场合，但在代码和游戏的世界里，我话还挺多的。

*   🔭 **正在折腾:** 用 Python 脚本自动化一切懒得动手的事情。
*   🌱 **正在学习:** Docker 容器编排和更底层的网络协议。
*   ⚡ **日常状态:** All in BOOM! (家里云维护员，致力于把服务器搞挂再修好)。
*   🎮 **游戏成分:** 《Minecraft》老矿工 & 《崩坏：星穹铁道》开拓者。
